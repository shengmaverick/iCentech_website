# iCentech Design System

## Purpose

This folder now acts as the single source of truth for the redesigned website's visual system.

这个目录现在是新版网站视觉系统的统一入口。

## Start Here

- Final color spec: `design-system/COLOR_SYSTEM.md`
- Shared tokens: `design-system/tokens.css`
- Generated implementation: `static-site/assets/site.css`

## Typography

- Body: `Inter` + `Noto Sans SC`
- Headings: `Space Grotesk` + `Noto Sans SC`
- Utility / metadata: `IBM Plex Mono`
- Source: Google Fonts
- License model: open-source, free to use, and suitable for public commercial websites

说明：
- 正文字体使用 `Inter` 搭配 `Noto Sans SC`
- 标题字体使用 `Space Grotesk` 搭配 `Noto Sans SC`
- 标签、导航元信息使用 `IBM Plex Mono`
- 以上字体均为公开、免费、合法可商用的开源字体

## System Notes

- The palette is anchored to the blue and green from the official iCentech logo.
- The page layout now follows a darker editorial direction while keeping the original site structure and menu structure.
- The site now supports both a dark editorial theme and a light white-base theme.
- Blog content is managed inside the repo and published into the new site, not linked back to the old platform.
